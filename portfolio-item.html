Project Description

Welcome to Step Rouge, a simple web based rhythm sequencer that I built using ReactJS, the Web Audio api, and the Web Midi api. To get started, simply select one of the 8 boxes in each of the columns. Each row of boxes corresponds to a musical note which is displayed on the left of the page. Note that only one note can be played per column. All you have to do now is press the PLAY button! Step Rouge also supports midi control (currently the only supported controller is the Novation launchpad and Launchpad Pro).
