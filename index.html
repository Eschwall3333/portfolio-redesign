Step Rouge

Welcome to Step Rouge, a simple web based rhythm sequencer that I built using ReactJS, the Web Audio api, and the Web Midi api. To get started, simply select one of the 8 boxes in each of the columns. Each row of boxes corresponds to a musical note which is displayed on the left of the page. Note that only one note can be played per column. All you have to do now is press The PLAY button!

Riddler's Clickable Guess

This is a simple yet woafully addicting card clicking game that I built with React.js. The idea is simple, each character from Batman's world of Gotham city has a random value assigned to it that changes everytime it's clicked. There's an unknown maximum value of card values that must not be exceeded or else you lose. The objective is to keep beating your high score.....what score can you reach?

The p5 Drummer

The p5 Drummer is a simple drum sequencer that I built using p5 editor. Simply click as many boxes as you want to create a drum pattern and press the "up" arrow on your key board to play and stop. Use the slider just below the arrangement grid to adjust for tempo. Enjoy!

Wifi Hero

Wifi Hero is a simple wifi hot spot search application built with vanilla JavaScript. This was a group project with a simple objective...find wifi hotspots with a decent signal based on your city or zipcode.
